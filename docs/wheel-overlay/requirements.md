# Requirements

## System Requirements

- **Windows 10 or later** (64-bit)

See [Common Setup](/common-setup/) for the full operating system setup guide shared across all OpenDash overlays.

## Hardware Prerequisites

- A **DirectInput-compatible sim racing wheel** with a rotary encoder, connected via USB
    - Verified compatible: BavarianSimTec Alpha
    - Other DirectInput wheels should be detectable; unsupported devices appear in the device list but may have limited position count defaults
- The wheel must be **plugged in before launching** WheelOverlay
